Here you can find a script which walks through game "Cragne Manor".

About the game: https://ifdb.org/viewgame?id=4x7nltu8p851tn4x

Download the game:

    wget https://ifarchive.org/if-archive/games/glulx/cragne.gblorb

Play using utilities:
    Gargoyle - for Linux: https://zoomadmin.com/HowToInstall/UbuntuPackage/gargoyle-free
    Spatterlight - for Mac: https://github.com/angstsmurf/spatterlight/releases
    Windows Frotz - for Windows: https://github.com/DavidKinder/Windows-Frotz/releases/

    https://rcveeder.net/cragne/play.html
