Demo of Google Test, from book "Modern C++ Programming with
Test-Driven Development" by Jeff Langr:

    https://www.amazon.com/Modern-Programming-Test-Driven-Development-Better-dp-1937785483

For details of integrating Google Test into CMake scripts, see:

    https://github.com/google/googletest/blob/master/googletest/README.md

How to build and run:

    mkdir build
    cd build
    cmake ..
    make
    ./test
